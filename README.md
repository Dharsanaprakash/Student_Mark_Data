# Student Data Cleaning using Excel

## Project Objective
Clean and standardize a raw student dataset containing inconsistent formats, missing values, and invalid entries.

## Dataset Workflow
1. Raw Dataset
2. Data Cleaning Process
3. Final Validated Dataset

## Cleaning Steps Performed
- Removed extra spaces using TRIM
- Standardized text using PROPER and LOWER
- Corrected categorical inconsistencies (Gender, Class, Section)
- Handled missing values
- Converted text values to numeric format
- Validated Age, Marks, Attendance ranges
- Standardized Phone Numbers
- Applied Data Validation rules

## Excel Functions Used
TRIM, PROPER, LOWER, SUBSTITUTE, VALUE, IF, AND, OR, LEN

## Files Included
- Dirty_Student_Data.xlsx → Original dataset
- Cleaning_Process.xlsx → Formula-based cleaning
- Final_Validated_Data.xlsx → Analysis-ready dataset

## Outcome
Converted messy raw data into structured, validated, analysis-ready dataset.
